# nicoanime_feed
ニコニ立体の新着をatomフィードにします。

## install
For Ubuntu 14.04 and apache

    cd /var/www 
    git clone https://github.com/sanadan/nico3d_feed.git
    cd nico3d_feed
    ./setup

設置したサーバーの nico3d_feed にアクセスすると、フィードが出力されます。

フィードは約1時間毎に更新されます。
